Delta Formula

Δ = Σ(human events × authority weight) / Σ(all event weights) = (1.0 + 0.8 + 0.5 + 1.0) / (1.0 + 0.0 + 0.8 + 0.5 + 1.0) = 3.3 / 4.5 = 0.73

The formula, event weights, and event hashes are included in the signed receipt payload. If the score or any event changes, the signature check fails.
